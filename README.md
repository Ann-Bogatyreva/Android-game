Тут наш не большой проект, разработка Android приложений на языке Котлин, мы делаем игру-пазл, основная задача самого приложения:
1. Игра-пазл
2. Уровни
3. Без зависаний и вылетов

Основная наша задача:
1. Сделать что-то играбельное
2. ± Оптимальное
3. Сделать переходы между "экранами" (возможность перехода от главного экрана, к выбору уровней и к уровням, а так же возможность возращения обратно)
4. Сделать на уровнях поля с детальками разделёнными и рандомно разбросанными по пространству уровня
5. Подобрать подходящие картинки для уровней
6. Нарисовать фоны и иконку приложения, нарисовать картинка для уровня (не обязательно)
7. Сделать возможность усложнения уровней путём большего деления картинки (3x3, 5x5, 7x7, 10x10)
